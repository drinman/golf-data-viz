# Benchmark Data

Static JSON files containing average golf statistics by handicap bracket.

## Data Sources
- **Arccos**: Published aggregate stats from 1.1T+ data points (via public reports and API)
- **Shot Scope**: 100+ performance stats by handicap band (publicly available)
- **USGA**: Handicap distribution and scoring statistics
- **Lou Stagner (@LouStagnerGolf)**: Published amateur data analyses

- All benchmarks are for **amateur** golfers, never PGA Tour
- Data should be updated periodically as new aggregate reports are published
- Bracket boundaries follow USGA handicap index ranges
